# mythtv-services-api
## Typescript Client library for mythtv services API
### Supports
* Backend APIs
  * Channel Service
    * GetChannelInfo
    * GetVideoSourceList
    * GetChannelInfoList
  * DVR Service
    * GetEncoderList
    * GetRecordedList
  * Myth Service
    * GetHostName
    * GetHosts
    * GetSetting
  * Video Service
    * GetVideoList
* Frontend APIs
  * GetActionList
  * GetStatus
  * SendMessage
  * SendAction
  * SendKey
  * PlayRecording
  * PlayVideo
